# Equora — Horse Riding Website

Luxury horse riding club website built with React and Vite.

## Features

- Multilingual website (EN / CZ / RU)
- Responsive modern design
- Horse profiles
- Riding experiences
- Booking section
- Photoshoot services
- Live deployment with Vercel

## Tech Stack

- React.js
- Vite
- CSS3
- React Router
- i18next
- Framer Motion
